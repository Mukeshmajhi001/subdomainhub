# SubDomainHub 🌐

A clean, neumorphic-styled subdomain portfolio hub — showcasing all subdomains and micro-projects under one roof.

---

## ✨ Features

- Neumorphism design system with smooth shadows
- Responsive layout (mobile, tablet, desktop)
- Animated card grid with scroll-triggered fade-up
- Locked / Upcoming card state for unreleased projects
- Mobile hamburger menu with smooth drawer animation

---

## 🛠 Tech Stack

| Layer      | Technology              |
|------------|-------------------------|
| Markup     | HTML5                   |
| Styling    | CSS3 (custom properties)|
| Scripts    | Vanilla JavaScript      |
| Icons      | Font Awesome 6.5        |
| Fonts      | Sora + DM Sans (Google) |

---

## 🚀 Setup

No build step needed — it's a static site.

```bash
# 1. Clone the repo
git clone https://github.com/Mukeshmajhi001/subdomainhub.git

# 2. Open in browser
open index.html
```

Or just drag `index.html` into any browser.
